# CRUD de Servicios de Streaming y Licencias Digitales

Este proyecto fue desarrollado en Java utilizando JDBC y MySQL.
Permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre servicios digitales.

## Ejemplos utilizados
- Netflix Premium
- Crunchyroll
- Microsoft 365

## Funcionalidades
- Crear servicios
- Listar servicios
- Actualizar precios
- Eliminar registros

## Tecnologías utilizadas
- Java
- Eclipse IDE
- MySQL
- JDBC

## Base de datos
Nombre: `streaming_db`  
Tabla: `servicio`

## Ejecución
Ejecutar la clase `Main.java` para probar el CRUD.
